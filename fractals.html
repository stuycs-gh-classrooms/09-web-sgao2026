Koch Curve

The Koch Curve consists of four Koch curves of the previous depth arranged with every interation of the previous depth forming one side of the Koch curve and a Koch curve depth of 1 being a straight, horizontal line


			koch(t, 4, 20)

Koch Curve Modification

The Koch curve with each curve as a side of a trangle and the angles inverted, known as a Koch snowflake, and at the end of every depth 1 curve, the turtle also draws a modification of the tree fractal at a random angle


			koch_spicy(child, 4, 20)
		

		def koch_mod(t, depth, length):
			if depth > 1:
				koch_mod(t, depth - 1, length)
				t.rt(60)
        			koch_mod(t, depth - 1, length)
        			t.lt(120)
        			koch_mod(t, depth - 1, length)
        			t.rt(60)
        			koch_mod(t, depth - 1, length)
    			else:
        			store = t.heading()
        			t.seth(random.randrange(361))
        			tree_spicy(t, 2, length // 2, 30)
        			t.seth(store)
        			t.color('black')
        			t.fd(length)

		def koch_spicy(t, depth, length): #koch snowflake + tree_spicy

			koch_mod(t, depth, length)
			t.rt(120)
			koch_mod(t, depth, length)
			t.rt(120)
			koch_mod(t, depth, length)

Sierpinski Triangle

The Sierpinski Triangle consists of 3 Sierpinski triangles of the previous depth--a depth 1 Sierpinski triangle being an equilateral triangle--arranged to form a larger triangle.


			sierpinski(t, 5, 200)

Sierpinski Triangle Modification

The width of the line increases as depth decreases and the color of the line depends on which side of the triangle the turtle is drawing.


			sierpinski_spicy(t, 5, 200)
		

		def sierpinski_spicy(t, depth, length): 
		    t.width(1)
		    if depth > 1:
		        sierpinski_spicy(t, depth - 1, length / 2)
		        t.lt(60)
		        t.fd(length / 2)
		        t.width(t.width() + 1)
		        t.rt(60)
		        sierpinski_spicy(t, depth - 1, length / 2)
		        t.rt(120)
		        t.fd(length / 2)
		        t.width(t.width() + 1)
		        t.lt(120)
		        t.fd(length / 2)
		        sierpinski_spicy(t, depth - 1, length / 2)
		        t.width(t.width() + 1)
		        t.bk(length / 2)
		    else:
		        t.lt(60)
		        t.color('cyan')
		        t.fd(length)
		        t.rt(120)
		        t.color('pink')
		        t.fd(length)
		        t.rt(120)
		        t.color('purple')
		        t.fd(length)
		        t.rt(180)

Tree

The tree consists of three Trees of the previous depth--with a depth 1 tree being a straight line--with one tree vertical and the other two at the angle given to the function.


			tree(child, 4, 50, 20)

Tree Modification

The Tree except there are flowers of a random color and size at the ends of every depth 1 tree.


		tree_spicy(child, 4, 50, 20)
	

		def tree_spicy(t, depth, length, angle): 
		    if depth > 1:
		        t.fd(length)
		        t.rt(angle)
		        tree_spicy(t, depth - 1, length, angle)
		        t.lt(angle * 2)
		        tree_spicy(t, depth - 1, length, angle)
		        t.rt(angle)
		        t.bk(length)
		        t.width(3)
